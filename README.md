# Sports Computer Vision Projects

Advanced Computer Vision and Sports Analytics projects built with Python, OpenCV, YOLO, Machine Learning, and 3D Visualization.

---

## Overview

This repository contains experimental Sports CV projects focused on extracting tactical and analytical insights from sports footage.

The goal is to demonstrate how modern computer vision systems can transform raw video into actionable intelligence through:

* Object Detection
* Multi-Object Tracking
* Ball Tracking
* Tactical Analysis
* Heatmaps
* Match Analytics
* 3D Projection & Visualization
* Sports Intelligence Dashboards

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/sports-cv-projects.git
cd sports-cv-projects
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate environment:

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running Projects

Football:

```bash
python football_intelligence.py --video match.mp4
```

Tennis:

```bash
python tennis_intelligence.py --video match.mp4
```
